HTML

HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets.

    Meta-Related Elements
  • Typically not seen by users, but it contains important information about accessibility, search engines, and performance.
  • The !DOCTYPE html declaration is letting the browser know to expect HTML on the page.
  • The HTML opening tag, represents the root, or start, of our HTML document.
  • Inside the html tag is an attribute of lang="en", which tells the browser the language for the webpage.
  • The title element defines the title for a webpage, which is is extremely important for Search Engine Optimization (SEO).
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport. It defines the charset, or character set, as UTF-8, which allows us to use most characters and symbols on the page.
  • The script element embeds client-side scripts in the HTML document.
    Content Division Elements
  • The div element creates a division in the page
  • The span element creates an inline division
  • NOTE: It is best practice to use semantic elements whenever possible, especially when the content is visible to users.
    Semantic Elements
  • Makes your HTML easier to read by other developers and the browser.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • Elements that are nested inside other elements are called child elements.
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items. In essence, these HTML elements represent bulleted or numbered lists.
  • The img element contains information about images that are displayed on the webpage. The `src` attribute defines the location of the image file and the `alt` attribute contains a text string that describes the appearance and functionality of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link.
  • The section elemtent defines a section in a document that contains thematic content.

CSS

There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.

    Inline CSS
  • Style an element directly in the HTML file.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Useful for quick and permanent changes but are less flexible than external and internal style sheets.
    Internal CSS Style Sheet
  • This internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • A single change to the CSS rule will apply to all tagged page elements.
    External CSS Style Sheet
  • Preferred, in most cases, because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • When using an external style sheet, we link the file using "link rel="stylesheet" href="./assets/style.css"
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • CSS Rule: Selector {property:value}
  • The margin property indicates how much space we want around the outside of an element.
  • The padding property indicates how much space we want around the content inside an element.
  • To apply a rule to all the elements visible on the page, use a special wild card symbol, or *.
  • The box-shadow CSS property adds shadow effects around an element's frame.
  • Assigning colors in CSS can be done in many ways, including by semantic references such as gray or blue. We can also refer to a color by its hexadecimal code, or hex #.

Git

Git commands always start with git and are used to save and track versions of your code.

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: returns to the main branch
  • git add -A: add change to the branch
  • git commit -m "message": commit to the repository (We use the -m flag to associate a message with our commit)
  • git push origin feature/branch-name: creates a PR in GitHub
  • git clone [SSH key]: clones repository

JavaScript

Is a high-level dynamic programming language that allows users to interact with websites.

  • A variable (var) is a named container that allows us to store data in our code.
  • “=” assignment operator
  • All strings are surrounded by quotation marks.
  • Each line of JavaScript code ends with a semicolon.
  • We can use the console.log method to output a message to the web console by adding an argument.
  • To add js to your application: go to your index.html file, just before the closing body tag, add 'script src="scripts/main.js"'
  • Control flow is the order in which a computer executes code in a script.
    Conditional Statements
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • An if statement is a way of introducing decision-making into our code. If the statement is truthy, then it will do whatever action we ask it to do in between the curly brackets.
  • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false.
  • If the statement is falsy, then the computer will ignore whatever is in between the curly brackets and move on.
  • The strict equality operator (===) checks to see if two values are equal.
    Loops and Arrays
  • Array = [ “”, “” ….]
  • Arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • for(var x = 0; x < shapes.length; x++) {the code that we want to be executed for each loop}
    Function
  • A function's name is followed by parentheses().
  • Special Math function helps to select a random number. Example: var randomTopic = topics[Math.floor(Math.random() * topics.length)]
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
